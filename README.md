# Reversi

![](demo.gif)

IF THERE IS NO OPENGL YOU HAVE TO INSTALL IT:
    
for apt:
    
        sudo apt-get install libglu1-mesa-dev freeglut3-dev mesa-common-dev

for pacman:

        sudo pacman -S glu freeglut

for macos it should already work fine
    
build it

    chmod +x build

    ./build

and run

    ./game

Enjoy Reversi!
